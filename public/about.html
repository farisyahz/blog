Awesome Website, How'd You Make It!??

I created this website from scratch using HTML, CSS with Tailwindcss plugin, and a little bit of Javascript. Then I simply deploy it using github pages. Tailwindcss really help me creating this website, otherwise it would be a hell of a work to style each element on this website. Tailwindcss allows you to write full css code inside of your HTML file. So it's like an inline CSS with steroids. This is Tailwindcss Documentations in case you want to try using it.
